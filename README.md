## Marionette HW Description

Marionette provides these perifherals:
* 60 GPIO
* 15 ADC Inputs
* 5 DAC Outputs
* 2 x I2C
* CAN 2.0B
* SPI
* SD Card

## Other
* Board is designed in EAGLE 7.x and are fabricated using OSH Park's 4 layer service
* BOM is available as an .XLSX file
